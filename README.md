# 🌡️ Temperature Converter

A responsive Temperature Converter web application built using HTML, CSS, and JavaScript as part of the Oasis Infobyte Web Development Internship (Level 1 - Task 3).

## 🚀 Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Responsive and user-friendly interface
- Input validation
- Instant conversion results

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
